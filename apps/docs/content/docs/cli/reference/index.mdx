Complete reference for all `supa` CLI commands.

## Commands

| Command | Description |
|---------|-------------|
| [`init`](/docs/cli/reference/init) | Initialize a new supabase project |
| [`dev`](/docs/cli/reference/dev) | Watcher that auto syncs changes to hosted environment [long-running] |
| [`orgs`](/docs/cli/reference/orgs) | List organizations |
| [`projects`](/docs/cli/reference/projects) | Manage projects |
| [`project`](/docs/cli/reference/project) | Project operations |
