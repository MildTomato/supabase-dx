---
title: "supa orgs"
description: "List organizations"
---

List all organizations you have access to.

Use this to find organization slugs for other commands like `supa init --org` or `supa projects list --org`.

## Usage

```bash
supa orgs [options]
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `--json` | boolean | Output as JSON |

## Examples

### List all organizations

```bash
supa orgs
```

### Get organizations as JSON

```bash
supa orgs --json
```
