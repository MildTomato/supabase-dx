## Usage

```bash
supa projects new [options]
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `--org <SLUG>` | string | Organization ID |
| `--region <REGION>` | string | Region (e.g., us-east-1) |
| `--name <NAME>` | string | Project name |
| `-y, --yes` | boolean | Skip confirmation prompts |
| `--dry-run` | boolean | Preview what would be created without making changes |
